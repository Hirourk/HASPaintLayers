Introduction

Welcome to the Blender Addon documentation! This guide will walk you through the features, installation process, and usage of the addon.

Installation

To install the addon:

  1. Download the addon ZIP file from the GitHub repository.
  2. Open Blender and go to Edit > Preferences > Add-ons.
  3. Click Install, select the downloaded ZIP file, and click Install Add-on.
  4. Enable the addon in the list.

Usage

After installation, you can access the addon features by navigating to:

  • Menu Path: Where to find it in the UI.
  • Hotkeys: Any shortcuts you can use.
  • Functionality: Detailed explanation of the addon’s functionality.

FAQ

Q: How do I update the addon?

A: Download the latest version from the GitHub repository and repeat the installation process.

Q: Is this addon compatible with all Blender versions?

A: This addon is tested with Blender 3.0 and above. Older versions may not be supported.
